## 项目概述

本项目是基于 Vue3 的纯前端商品交易系统。系统提供用户注册、登录、购物车以及后台管理等功能模块，适用于快速构建商城类应用的前端部分。

## 功能模块

### 用户模块

- 用户注册
- 用户登录

### 购物车模块

- 商品添加
- 商品删除
- 购物车结算

### 后台管理模块

- 商品管理
  - 添加商品
  - 编辑商品
  - 删除商品

## 系统角色

- 普通用户：负责浏览商品、注册、登录、购物等操作。
- 管理员：负责后台的商品管理。

## 运行环境

- Node.js 环境
- npm 或 yarn 包管理工具
- 支持 HTML5 和 CSS3 的现代浏览器

## 部署步骤

1. 安装 Node.js 和 npm 或 yarn
2. 克隆项目代码到本地
3. 在项目根目录下运行 `npm install` 或 `yarn` 安装依赖
4. 运行 `npm run serve` 或 `yarn serve` 启动开发服务器

## 目录结构

```
project/
├── public/               # 公共文件目录
├── src/                  # 源代码目录
│   ├── assets/           # 静态资源
│   ├── components/       # Vue 组件
│   ├── views/            # 页面级Vue组件
│   ├── router/           # 路由配置
│   ├── store/            # 状态管理
│   └── App.vue           # 主组件
│   └── main.js           # 入口文件
├── package.json          # 项目描述文件
```
